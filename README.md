## About

Sonnatech (صُنّاع تك) is an Egyptian software company focused on building smart, accessible tools for local businesses and digital payment agents. We design systems that solve real problems — fast, reliable, and built for the Egyptian market.

## Products

### 🟢 Geebi — Cafe POS System
> **The complete point-of-sale solution for Egyptian cafes**

[geebi.sonnatech.info](https://geebi.sonnatech.info)

Geebi helps cafe owners streamline operations with a mobile-first POS system built for Egypt. From order management to real-time analytics, it covers everything a modern cafe needs to run efficiently and grow.

**Key features:**
- Mobile POS app (iOS & Android) with offline support
- Real-time sales analytics and business dashboards
- Multi-branch management and cloud sync
- Smart inventory tracking with low-stock alerts
- Support for Vodafone Cash, Fawry, InstaPay, Visa, and more
- Arabic and English support with 24/7 customer service
- RESTful API for custom integrations

---

### 🟣 Paylo (أدفعلو) — Digital Wallet Management
> **Smart management system for Egypt's digital payment agents**

[paylo.sonnatech.info](https://paylo.sonnatech.info)

Paylo is built for wallet agents (Vodafone Cash, Orange Money, Etisalat, WE, InstaPay) who need a reliable way to track transactions, calculate commissions automatically, and manage their team — all from one place.

**Key features:**
- Automatic commission calculation per wallet and transaction type
- Real-time and historical reports (daily, weekly, PDF/CSV export)
- PIN-protected security for sensitive operations
- Team management with role-based access control
- Offline sync — works without internet, syncs automatically when reconnected
- Referral program: earn 100 EGP per referred subscriber
- Flexible plans: monthly (299 EGP), annual (2,799 EGP), and 3-year founder plan
